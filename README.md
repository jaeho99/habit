배워야하는 또는 배우고 싶은 기술 나열하기
React
정의 : 사용자 인터페이스(UI) 개발을 위한 자바스크립트 라이브러리
장점 :
높은 성능 : 가상 DOM(Virtual DOM)을 사용하여 변경된 부분만 효율적으로 업데이트하므로, 순수 자바스크립트나 제이쿼리보다
빠르고 부드러운 화면 전환이 가능하다.
코드 재사용성 및 유지보수성 : 컴포넌트 단위로 코드를 관리하여 코드의 재사용성이 높고, 애플리케이션 구조가 깔끔해져 유지보수가 용이
뛰어난 유연성 : 웹 개발뿐만 아니라 React Native를 통해 iOS/안드로이드 네이티브 앱을 개발하거나 일렉트론을 통해 데스크톱 앱도 개발할 수 있는 등 확장성이 뛰어나다.
강력한 커뮤니티와 생태계 : 활발한 커뮤니티와 방대한 자료, 숙련된 개발자들이 많아 문제 해결과 기술 지원을 받기 용이하다.

Thymeleaf
정의 : Java 기반의 현대적인 서버사이드 템플릿 엔진으로, HTML, XML, JavaScript, CSS 등 다양한 형식의 템플릿을 처리할 수 있다.
주요 기능 : 컨트롤러에서 전달된 데이터를 HTML 파일을 열어 화면을 미리 볼 수 있다.
장점 :
HTML 표준을 준수하여 서버 구동 없이도 HTML 파일을 열어 화면을 미리 볼 수 있다.
th 속성을 사용해 동적인 데이터를 처리하면서도 HTML의 기본 구조를 유지할 수 있어 디자이너와의 협업이 용이하다.
활용 : 주로 Spring Boot와 같은 Java 기반 웹 애플리케이션에서 사용된다.

정처기 코드 문제 풀이하면서
람다식을 왜 사용하는지 람다식의 사용 방법에 대하여 궁금해졌다.
